- 训练样本: 30场
- ECE (期望校准误差): 0.128

### ⚠️ 校准粗粒度警告

经诊断发现，当前校准器存在**粗粒度映射问题**：30个训练样本产生的Isotonic回归台阶函数分辨率有限，在概率中段(0.35-0.55)，多个不同输入被映射到完全相同或非常接近的输出值。

具体验证:
```
输入: 土耳其 51.2% / 巴拉圭 26.0%  → 校准输出: 59.4% / 13.0%
输入: 捷克   36.6% / 南非   39.8%  → 校准输出: 59.4% / 13.0%  ← 完全一致!
输入: 巴西   64.3% / 海地   17.0%  → 校准输出: 66.8% / 10.6%  ← 仅轻微差异
```

这意味着校准器在概率中段几乎将所有输入都映射到 ~59%/28%/13% 的分布。**这不是代码Bug，而是30样本下Isotonic回归的固有限度** — 校准曲线台阶太少，无法区分精细的概率差异。

**对本次预测的影响**: 校准后的59.4%-27.6%-13.0%应被视为"校准器认为的合理中段分布"，而非针对土耳其vs巴拉圭的精准校准。建议同时参考未校准的最终概率(51.2%-22.8%-26.0%)作为对照。
