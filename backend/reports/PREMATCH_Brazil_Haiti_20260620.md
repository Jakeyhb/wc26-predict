### 3.2 模型-市场分歧: 🔴 38.0pp (极端)

```
模型 Pre-Market: 巴西 49.1% | 平局 25.3% | 海地 25.6%
市场隐含概率:    巴西 87.1% | 平局  8.7% | 海地  4.2%
分歧:            巴西 38.0pp | 平局 16.6pp | 海地 21.4pp
```

**这是继捷克vs南非(31.7pp)之后更大的模型-市场分歧！** 市场将巴西定价为1.10的极端热门（隐含87%胜率），而模型融合后仅给出49%的巴西胜率。

**分歧原因诊断**:

1. **DC模型的结构性低估**: DC基于俱乐部数据训练，巴西国家队的真实实力被俱乐部参数稀释。巴西atk=1.86远低于顶级俱乐部水平（如曼城>2.5），但在国家队层面，巴西的实际进攻火力远超此数值。

2. **Pi模型是唯一的"清醒者"**: Pi给出巴西85.8%的胜率，与市场87.1%几乎一致。Pi可能捕捉到了DC和Enhancer忽略的实力维度。

3. **Enhancer对海地的过度乐观**: 海地30.9%的胜率（Enhancer）可能来自世预赛对弱旅的优异表现，但世界杯正赛是完全不同的级别。

4. **Elo差距被压缩**: 巴西1754 vs 海地1580，174分的差距远比真实实力差距小。如果使用更准确的Elo数据（巴西可能在1950+，海地在1450-），Elo会给出>85%的巴西胜率。

### 3.3 动态市场权重增强 🔧

根据V3.9.5.3机制:
```
分歧 38.0pp > 15pp 阈值 → 触发动态市场增强
增强幅度 = min(0.12, (0.38 - 0.15) × 0.8) = 0.12 (达到上限)
market_weight: 0.28 → 0.40 (上限)
```

**融合后 (Post-Market)**:

| 指标 | 数值 |
|------|------|
| 巴西胜 | **64.3%** |
| 平局 | 18.7% |
| 海地胜 | 17.0% |

市场权重从28%提升到40%后，巴西胜率从49%→64%，海地从26%→17%。但即使是增强后，模型给出的巴西胜率仍远低于市场的87%。

### 4.1 ⚠️ 校准粗粒度警告

**本节为全局自审后追加的重要披露。**

经诊断发现，当前校准器存在**粗粒度映射问题**：30个训练样本产生的Isotonic回归台阶函数分辨率有限，在概率中段(0.35-0.70)，多个不同输入被映射到完全相同或非常接近的输出值。

具体验证:
```
输入: 巴西   64.3% / 海地   17.0%  → 校准输出: 66.8% / 10.6%
输入: 土耳其 51.2% / 巴拉圭 26.0%  → 校准输出: 59.4% / 13.0%
输入: 捷克   36.6% / 南非   39.8%  → 校准输出: 59.4% / 13.0%  ← 几乎一致!
```

巴西的输入(64.3%)落在校准曲线的较高区域，输出(66.8%)略有拉升。但校准器的台阶分辨率仍不足以对64.3%和51.2%的输入做出足够精细的区分。**校准后的66.8%/22.6%/10.6%不应被视为针对本场的精准校准，而应理解为"校准器认为的中段合理分布"**。建议同时参考未校准的Post-Market概率(64.3%/18.7%/17.0%)作为对照。

| 海地无压力 | 海地↑ | 世界杯首秀，面对巴西毫无包袱，可能超常发挥 |

### 8.3 模型-市场分歧的结构性解读

38.0pp的分歧不应简单理解为"市场错了"或"模型错了"：

- **市场(87%巴西)的逻辑**: 巴西是五届世界杯冠军、足球王国；海地是世界杯新军、世界排名100+。在实力论面前，87%的胜率"感觉合理"。
- **模型(49%巴西)的逻辑**: 数据驱动的DC/Enhancer看到的是巴西并非不可战胜（首战1-1摩洛哥），海地防守并不差（0-1苏格兰），xG差距不大（1.40 vs 0.92）。

**最可能的情况**: 真实概率在两者之间。校准后的66.8%是当前系统的最佳估计，但存在±10pp的不确定性。如果巴西的真实实力强于DC参数显示的水平（大概率如此），那么70%+的胜率也是合理的。
